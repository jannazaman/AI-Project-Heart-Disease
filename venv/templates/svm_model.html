Kernels

  • Linear Kernel: A simple kernel that creates linear decision boundaries, works well for linearly separable data.
  • Tanh (Hyperbolic Tangent): Outputs values in the range [-1, 1], capturing non-linear relationships.
  • Linear Kernel: A simple kernel that creates linear decision boundaries, works well for linearly separable data.

Model Advantages

  • SVM can efficiently handle non-linear decision boundaries using different kernel functions.
  • SVM is less prone to overfitting, especially in high-dimensional spaces.

Model Disdvantages

  • Training an SVM model can be computationally intensive, especially for large datasets.
  • SVMs can be sensitive to noise in the training data, especially when there is a high degree of overlapping between classes.
